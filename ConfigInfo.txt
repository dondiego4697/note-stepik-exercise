Окружение:

1. Swift Compiler - Custom Flags -> Other Swift Flags
На Debug и Release проставлены соответствующие флаги -D DEBUG и -D RELEASE.
2. В Libs/Env.swift есть код проверяющий на production окружение.
3. При запуске, в схеме можно поменять Build configuration и отработает соответсвующий флаг

Использовать можно для любых целей:
1. Проверка на необходимость вывода в консоль.
2. Засчет статистики только в проде
3. Разные backend хосты
